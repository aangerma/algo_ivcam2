AlgoCameraCalibration (ACC) - post swap calib tool. Nowadays replaced by SW CAL.
AlgoThermalCalibration (ATC) - post swap thermal calib tool. Nowadays replaced by SW CAL THERMAL.
IV2ThermalCalibTool (Algo2) - pre swap thermal calib tool. Nowadays used as thermal validation tool (Algo2Val).
IV2valTool (AV) - independent validation test.
